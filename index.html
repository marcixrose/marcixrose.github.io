CSS Font Styling


There are a lot of ways to stylize fonts for your website. Here are some commonly used styling properties.


@font-face This allows you to import custom fonts
directly into your CSS stylesheet.
font-family This is how you select certain fonts for
different elements within your CSS stylesheet.
font-style This allows you to change the style
of the font, i.e. italic, bold, light.
font-size This allows you to increase or decrease
the size of your font per element.
